Getting Started

Installation and First Launch

Digital Workshop is a comprehensive 3D model management application designed for CNC woodworking and digital fabrication. When you first launch the application, you'll see the main window with several dockable panels and a central tab area.

[Image: main-window-overview.png - Main application window showing all panels]

System Requirements

  • Windows 7 or higher
  • Python 3.8+ (included with installer)
  • 8GB RAM minimum (16GB recommended)
  • OpenGL 3.3 compatible graphics card
  • 500MB free disk space for installation

Command Line Options

Digital Workshop supports several command-line options for advanced users:

  • --debug or --log-level DEBUG - Enable verbose debug logging
  • --log-console - Enable console logging in addition to file logging
  • --log-human - Use human-readable log format instead of JSON
  • --mem-only - Run database in memory only (development mode)
  • --nuke - Nuclear reset: destroy ALL application data after 5-second countdown
  • --nuke-no-backup - Nuclear reset without creating backup
Note: The nuclear reset feature is irreversible and will delete all your models, settings, and database. Use with caution.

Layout Edit Mode

Digital Workshop features a Layout Edit Mode that allows you to customize your workspace:

  • Layout Locked (Default) - Docks are locked in place, preventing accidental movement
  • Layout Edit Mode - Docks can be moved, resized, and rearranged

To toggle Layout Edit Mode, use the menu option View → Layout Edit Mode or the toolbar button.

Pro Tip: The application always starts with Layout Edit Mode OFF to prevent accidental changes to your carefully arranged workspace.

Troubleshooting

Common Issues and Solutions

Performance Issues

Issue: Application runs slowly with large models

Solution: Enable adaptive quality in preferences and reduce model complexity

Issue: High memory usage

Solution: Close unused applications and enable progressive loading

Display Problems

Issue: 3D viewer appears black or corrupted

Solution: Update graphics drivers and ensure OpenGL compatibility

Issue: Materials not applying correctly

Solution: Check model format compatibility and verify material files

Import/Export Issues

Issue: Models fail to import or export

Solution: Verify file permissions and check disk space

Issue: Duplicate models in library

Solution: Use duplicate detection and management tools

AI Analysis Problems

Issue: AI analysis fails or returns errors

Solution: Check API key validity and internet connection

Issue: Slow AI response times

Solution: Enable caching and choose faster AI models
