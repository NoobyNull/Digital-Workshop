Digital Workshop - 3D Model Viewer and Editor

Version: 1.0.0

OVERVIEW
========

Digital Workshop is a modern, high-performance 3D model viewer and editor
designed for Windows. It provides a comprehensive toolkit for viewing, analyzing,
and working with 3D models in various formats.

FEATURES
========

- Support for multiple 3D file formats (STL, OBJ, STEP, 3MF)
- Real-time 3D visualization with VTK rendering
- Material and texture support
- Thumbnail generation and caching
- Metadata editing and management
- Performance monitoring and optimization
- Offline-first architecture with local data storage
- Modern Windows desktop integration

SYSTEM REQUIREMENTS
===================

- Windows 10 or later
- 2GB RAM minimum (4GB recommended)
- 500MB free disk space
- Graphics card with OpenGL support

INSTALLATION
============

1. Run the Digital Workshop installer
2. Follow the installation wizard
3. Choose installation directory (default: Program Files)
4. Complete the installation
5. Launch Digital Workshop from Start Menu or Desktop shortcut

GETTING STARTED
===============

1. Open Digital Workshop
2. Use File > Open to load a 3D model
3. Use the 3D viewer to interact with the model:
   - Left mouse button: Rotate
   - Right mouse button: Pan
   - Mouse wheel: Zoom
4. Access preferences for customization
